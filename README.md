A web application for discovering and recommending cocktails based on your mood and preferences. Built as a school hackathon project.

## Overview

Cocktail Maker is an interactive web app that helps users find the perfect 3-drinks recommendations while watching a show or just enjoying their time. Refresh will flip through curated cocktail selections and explore ingredients for each drink. A new batch of 3-drinks is a refresh of a page away!

Connect, compare, and craft cocktails instantly—thanks to our open‑API recipe engine. (An ingredients' list is on a back of the flash-cards)

## Features

-  **Show-Paired Recommendations** - Get cocktail suggestions tailored to different moods and activities
-  **Ingredient Lists** - View detailed ingredients for each cocktail recommendation
-  **Responsive Design** - Fully responsive layout that works on all devices
-  **Beautiful UI** - Modern, clean interface with smooth interactions

## Tech Stack

- **HTML5** - Structure and semantic markup
- **CSS/SCSS** - Styling and responsive layouts
- **JavaScript** - Interactive functionality
- **Axios** - API requests for cocktail data

## Project Structure

school-hackathon/ 
- index.html # Main entry point 
- scripts/script.js # Main functionality 
- keys.js # API configuration 
- styles/styles.css # Stylesheet (compiled from SCSS) 
- assets/  # Social media and UI icons
